Features

Size is Everything

Calculates sizes for all files & folders in the current directory. Sort by size to identify large entries.

Text-Based UI

No X11 or web server required. Liberator uses a text-based interface that works perfectly over SSH.

Quick Delete

Found something taking up too much space? Easily delete it without ever having to leave the app.

Usage

Key Description
j Select the next entry in the list.
k Select the previous entry in the list.
<enter> Enter the selected directory.
h Navigate to the parent directory.
s Sort the entries by size.
x Delete the selected entry.
q Quit the application.

Purge takes its interface inspiration from Vim, using (almost exclusively) the home-row keys to control its modal interface.
