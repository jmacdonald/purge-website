Quickly expel large files & folders from your filesystem.

Features

Size is Everything

Liberator calculates sizes for all of your files & folders, and can present them sorted by size.

Text-Based UI

No X11 or web server required. Liberator uses a text-based interface that works perfectly over SSH.

Quick Delete

Found something taking up too much space? Delete it without ever having to leave the app.

Usage

Liberator takes its control cue from Vim, using the home keys to navigate your filesystem.

Use j, k, h, and enter to move up, down, out, and into directory entries.
Use x to delete the selected entry.
